About This App

Back Menu

This cross-platform mobile app has been developed using HTML5, CSS3 and Javascript technologies. The tools or frameworks used for the development are Apache Cordova (or PhoneGap) and jQuery Mobile. Cordova has been used for implementing access to native features and jQuery Mobile has been used as UI framework. The application was built with the help of 'PhoneGap Build'.

This mobile app test some of the mobile device features such as Accelerometer, Camera, Geolocation, and more. Also, the client-server architecture has been implemented for login and some other features.

This mobile app has been developed as a part of my dissertation project for MSc. in Software Engineering.
